C2nxt provides an assortment of platform detection macros to detect operating system, compiler, little vs big endianness, and architecture.

Note

while C2nxt provides many different platform detection macros, it has currently only been tested on x86_64 Windows, x86_64 linux, and ARM64 (M1) MacOs
