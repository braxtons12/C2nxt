#define Ok(T, value)
Creates a StdResult(T) holding the given value.

Creates a StdResult(T) holding the given error.

Returns whether this StdResult(T) is holding a value.

Returns whether this StdResult(T) is holding an error.

#define std_result_and(self, result_b)
Returns result_b is self is Ok. Otherwise returns the Err value of self.
#define std_result_and_then(self, next_func)
Returns the result of calling next_func with the contained value if self is Ok. Otherwise, returns the Err value contained in self
#define std_result_or(self, result_b)
Returns self if it is Ok, otherwise returns result_b
#define std_result_or_else(self, func)
Returns self if it is Ok, otherwise returns the result of calling func
