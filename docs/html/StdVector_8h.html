git/C2nxt/include/C2nxt/StdVector.h file

This module provides a dynamic-array type comparable to C++'s std::vector and Rust's std::vec::Vec for C2nxt.

#define std_vector_resize(self, new_size)
Resizes the given StdVector(T) to new_size number of elements. If new_size is greater than the current size, this will allocate memory if necessary and default-construct new elements. If new_size is less than the current size, this will destruct size - new_size number of elements and, if new_size is less than the SSO capacity, deallocate memory.
