A StdStringView is a "view" into a StdString. It allows you to lazily inspect the contents of its associated string, but not modify them. It is effectively a fat reference to the contents of its associated string.

Classes

struct StdStringView
C2nxt stringview type. A stringview is a "view" into a StdString. It allows you to lazily inspect the contents of the string, but not modify them. It is effectively a fat pointer into the contents of the associated StdString
struct StdStringViewIterator
C2nxt stringview const iterator storage type StdStringViewIterator is the underlying storage type used by StdStringView for its const iterator type (StdRandomAccessIterator(const_char_ptr))
