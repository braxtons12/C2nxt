Cnx provides abstractions and ergonomics features for dealing with memory allocation, deallocation, and handling custom allocators

struct CnxAllocator
A CnxAllocator is a convenient abstraction for wrapping memory allocation and deallocation, making it simple and easy to write allocator-configurable and/or allocator aware software.
